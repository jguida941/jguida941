Governed control plane

Compose a design language: pick a base, then swap any component to another language. This is the design surface — separate from the builder scorecard.

An invalid combination is unconstructable. A swap is admissible ONLY if the composed component, as rendered, is a FULL valid instance of some active design language — a partial mix that belongs to no language (e.g. a frosted material on a square 0-radius card) is refused. The verdict below is computed by ONE Python source (scripts/quality/settings_admissibility.py), routed through the real render → conform path; this page only displays it (no verdict JS).
